Open the app

  • Launch Smart Time & Date Calculator.
  • Use the navigation drawer to choose a tool.
Screenshot coming soon

Use Time Forecast

  1. Select Time Forecast from the menu.
  2. Tap any time input field to open the custom keypad.
  3. Enter: Current AF / ENG hours, Life Expired (L.E), Installed at AF / ENG hours, Inspection/servicing due at AF / ENG hours.
  4. (Optional) Enter Component Name and Tag.
  5. Tap Calculate.
  6. Results appear instantly: current component hours, life remaining, inspection/servicing due at AF / Eng hours.
  7. Press the history icon in the top bar to view time forecast history.
Screenshot coming soon

Use Date Forecast

  1. Select Date Forecast.
  2. Choose the base date by radio: Manufacture date OR Installed date.
  3. Enter the base date.
  4. Enter Inspection/servicing due: numeric value (supports decimals), unit (Years / Months / Weeks / Days).
  5. (Optional) Enter Component Name and Tag.
  6. Tap Calculate.
  7. Results: inspection/servicing due on (date), life remaining (years/months/days).
  8. Use the history icon for date forecast history.

Input rules & notes

  • Required fields show validation via toast messages.
  • Time inputs use a custom keypad (no system keyboard).
  • Date inputs format automatically as DD/MM/YYYY.
  • Invalid dates are rejected and cleared automatically.

Navigation & UI tips

  • Drawer menu gives access to all calculators.
  • Top bar history icon changes based on current screen.
  • Back navigation may show a confirmation dialog (when configured).
